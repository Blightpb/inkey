Tecnologias utilizadas

Java

Java é uma linguagem de programação usada para criar softwares, aplicativos e sistemas. É conhecida por ser rápida, segura e funcionar em diferentes plataformas.

JavaScript

JavaScript é uma linguagem usada para tornar páginas web interativas. Ela controla ações dinâmicas no navegador, como animações, cliques, menus e muito mais.

HTML e CSS

HTML é a linguagem usada para estruturar o conteúdo de uma página (textos, imagens, botões). CSS é usado para estilizar esse conteúdo (cores, fontes, layout).
